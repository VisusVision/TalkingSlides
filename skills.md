# skills.md — Required skills & short descriptions

## Core backend
- Python (3.10+): async programming, packaging, virtualenv, pip.
- Django: models, admin, authentication, file handling.
- FastAPI: building ML/TTS microservices and typed APIs.
- Celery + Redis: task queuing and background workers.
- PostgreSQL: relational DB design and migrations.

## ML & audio
- PyTorch: model loading & inference for TTS models.
- TTS toolkits: familiarity with Coqui, Fish-Speech or similar.
- Audio tooling: ffmpeg, pydub, librosa, sox basics.
- Voice cloning concepts: zero-shot vs fine-tuning.

## DevOps & infra
- Docker & docker-compose; Dockerfile best practices.
- GPU containers: nvidia runtime basics, CUDA drivers.
- Cloud basics: object storage (buckets), CDN, VM instances.
- CI/CD: GitHub Actions or equivalent.

## Frontend & UX
- React (basic): components, forms, file uploads.
- Responsive UI and accessibility for educational content.
- Basic CSS frameworks & component libraries (design systems).

## Testing & quality
- pytest for unit/integration tests.
- Basic performance testing for worker throughput.
- Logging, monitoring, and error reporting.

## Soft skills
- Code review discipline & git branching model.
- Documentation & README writing for onboarding.
- Security and privacy awareness for handling audio data.
